# External Resources

## Essential Reading

### Books
- **"Designing Data-Intensive Applications"** — Martin Kleppmann. The system design bible. More relevant than ever because AI handles implementation but not design.
- **"The Pragmatic Programmer"** — Hunt & Thomas. Timeless principles about thinking clearly about software.

### Documentation
- [Claude Code Documentation](https://docs.anthropic.com/en/docs/claude-code) — Official docs, always up to date
- [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) — The protocol for extending AI agents
- [Anthropic Cookbook](https://github.com/anthropics/anthropic-cookbook) — Practical examples of using Claude

### Papers (Accessible)
- "Attention Is All You Need" (2017) — The transformer paper. Dense but foundational.
- "Constitutional AI" (2022) — How Claude is trained to be helpful and safe.
- "Chain-of-Thought Prompting" (2022) — Why "let's think step by step" works.
