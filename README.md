# AI-Native Coding: A Comprehensive Learning Project

> From understanding how LLMs work to orchestrating multi-agent systems that code while you sleep.

## Why This Project Exists

AI-native coding is not "coding with autocomplete." It is a fundamental shift in how software gets built — where the human becomes an **architect, reviewer, and decision-maker**, and AI agents handle implementation at scale. This project is your structured path to mastering that shift.

## How to Use This Project

1. **Read modules in order** — each builds on the previous
2. **Do the labs** — theory without practice is forgettable
3. **Keep a learning journal** — use `journal/` to capture insights, failed experiments, and breakthroughs
4. **Update this project** — AI tooling evolves weekly; treat this as a living document
5. **Teach what you learn** — best way to solidify understanding

## The Evergreen Mindset

This project is designed to grow with you. The `references/` folder contains pointers to external resources. The `journal/` folder is your personal changelog. Every time you discover a new technique, pattern, or tool — add it here. Your future self will thank you.
